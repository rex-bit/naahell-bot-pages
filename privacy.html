NAAHELL Bot

Privacy Policy

Last updated: May 2025

1. Information We Collect

NAAHELL Bot collects only the minimum data necessary to operate its features:

  • Discord User IDs — used to track quiz points and reward history
  • Discord Server (Guild) IDs — used to configure server-specific settings
  • Discord Channel IDs — used to manage quiz channels

We do not collect names, email addresses, passwords, or any other personal information.

2. How We Use Your Data

Data collected is used solely to operate the Bot's features, including:

  • Maintaining quiz point leaderboards
  • Tracking Robux reward history
  • Managing quiz session channels

3. Data Storage

All data is stored locally on the Bot's host server in JSON format. Data is not transmitted to or shared with any external services or third parties.

4. Data Retention

Data is retained for as long as the Bot is active in your server. Server administrators may request data deletion by contacting the Bot administrator directly via Discord.

5. Third-Party Services

The Bot interacts with the following third-party APIs to provide its features:

  • Roblox API — to retrieve game and group information
  • Discord API — to operate within the Discord platform

Use of these services is subject to their respective privacy policies.

6. Contact

For any privacy-related inquiries or data deletion requests, please contact the Bot administrator via Discord.

For questions, contact the Bot administrator via Discord.
